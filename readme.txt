 Plateforme Freelance

 Description
Cette plateforme permet aux freelances de consulter des missions disponibles et d'y postuler. Elle propose une interface moderne et interactive utilisant HTML, CSS et JavaScript, avec un stockage simulé des données en JSON.

 Fonctionnalités
- Affichage dynamique des missions à partir d'un fichier JSON.
- Formulaire de candidature permettant aux freelances de postuler.
- Tableau de bord des candidatures listant les candidatures soumises.
- Suppression des candidatures avec un bouton dédié.
- Interface responsive et ergonomique.

 Technologies utilisées
- HTML : Structure de la page web.
- CSS : Stylisation et mise en page.
- JavaScript : Gestion dynamique des données et interactions.
- JSON : Simulation d'une API REST pour les missions.

 Installation et exécution
1. Cloner le projet
   ```bash
   git clone https://github.com/selmib30/CDA_08.git
   cd plateforme-freelance
   ```
2. Ouvrir le projet dans un navigateur
   - Si vous utilisez VS Code, ouvrez `index.html` avec Live Server.
   - Sinon, ouvrez directement le fichier `index.html` dans un navigateur.

 Structure du projet
```
plateforme-freelance/
│── index.html           Page principale
│── style.css            Styles de l'interface
│── script.js            Gestion des interactions
│── data/
│   └── missions.json    Fichier contenant les missions simulées
└── README.md            Documentation
```

 Utilisation
1. Consulter les missions affichées sur la page d'accueil.
2. Remplir le formulaire en indiquant son nom et la mission choisie.
3. Soumettre la candidature, qui sera ajoutée au tableau de bord.
4. Supprimer une candidature si nécessaire en cliquant sur le bouton "Supprimer".

 Améliorations possibles
- Intégration d'une vraie API backend.
- Ajout d'un système d'authentification.
- Gestion avancée des candidatures avec persistance des données.
